我需要将写好的代码通过Git提交到某个服务器上去保存
所以需要使用Git做一些基本控制！ 


   git init // git 项目初始化 
   git status // 查看文件状态（保存或没保存） 
   git add . // 添加所有文件 
   git add 文件名 // 添加单个文件 
   git commit -m "描述信息" // 提交到本地仓库
   git restore 文件名 // 还原到上次提交之前 
   git init // git 项目初始化
